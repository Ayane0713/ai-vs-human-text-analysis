# AI vs Human Text Analysis
Exploratory data analysis comparing AI-generated and human-written texts.

## Overview

This project investigates whether linguistic features can distinguish AI-generated text from human-written text. The goal is to explore how statistical and linguistic patterns differ between the two and assess whether these differences can be used for classification.

---

## Tools

* Python (pandas, matplotlib)
* Jupyter Notebook

---

## Data

The dataset used in this project is sourced from Kaggle:

**AI Generated vs Human Written Text Dataset (2026 Edition)** by Muhammad Shahzad.

This dataset contains 1,000 structured text samples with engineered linguistic and statistical features for detecting AI-generated content. Features include:

* perplexity score
* lexical diversity ratio
* syntactic variability
* semantic coherence score

The dataset is included in the `data/` folder for reproducibility.

---

## Key Steps

* Loaded and explored the dataset
* Cleaned and prepared the data
* Conducted exploratory data analysis (EDA)
* Compared features between AI-generated and human-written text
* Applied statistical analysis to evaluate differences

---

## Key Findings

* Some linguistic features show differences between AI-generated and human-written text
* However, the differences are relatively subtle and overlapping
* This suggests that distinguishing AI-generated text based only on surface-level features can be challenging
